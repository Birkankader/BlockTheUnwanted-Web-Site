Frequently Asked Questions

Everything you need to know about Block The Unwanted

Is my data safe and private?

Yes. Free tier processing happens entirely locally in your browser using Tesseract.js. Premium tier only sends necessary content to OpenAI for analysis. We never store your personal data, browsing history, or filtered content.

How does automatic user blocking work?

The extension analyzes user behavior patterns and automatically blocks accounts that frequently post content matching your filter criteria. You can set custom thresholds for how many violations trigger an auto-block, and the system learns from spam patterns.

Does it work on both Twitter and X?

Yes, the extension works seamlessly on both twitter.com and x.com domains. It automatically adapts to interface changes and updates, ensuring consistent filtering across all Twitter/X versions.

What's the difference between Free and Premium tiers?

Free tier offers comprehensive local text and image processing with Tesseract OCR, automatic user blocking, and all core features. Premium tier adds OpenAI GPT-4o powered advanced OCR, video speech-to-text analysis, and higher accuracy detection (95%+ vs 85%+).

When will Premium features be available?

Premium features including OpenAI GPT-4o OCR and Whisper video analysis are currently in development. We're working hard to bring you AI-powered filtering capabilities. Follow our updates for launch announcements and beta testing opportunities.

Can I filter specific communities?

Yes, you can block entire Twitter communities by name, URL, or hashtag. The extension will hide posts from those communities, their members, and related content. It also supports automatic detection of community-related posts.

Does it slow down Twitter/X?

No. The extension is optimized for performance with efficient Service Worker architecture, smart caching, and minimal memory usage. Most users see no noticeable impact on browsing speed, and some report faster loading due to ad blocking.

Can I temporarily disable filtering?

Yes, there's a master toggle in the extension popup that instantly enables/disables all filtering. You can also toggle individual filter types (text, OCR, communities, ads, user analysis) independently for granular control.

What languages does the OCR support?

The local Tesseract OCR supports major languages including English, Turkish, Spanish, French, German, Arabic, and more. Premium tier OCR with OpenAI GPT-4o supports virtually all languages with higher accuracy and context understanding.

Can I export my filter settings?

Yes, you can export your keywords, communities, blocked users, and all settings as a JSON file for backup or transfer to another device. You can also import settings from other users or restore from backups.

Is there a mobile version?

Currently, this is a Chrome desktop extension. Mobile browser extensions have limited capabilities, but we're exploring mobile solutions including potential native apps for the future. Stay tuned for updates!

How accurate is the automatic spam detection?

The automatic user behavior analysis has a 95%+ accuracy rate in detecting spam and problematic accounts. The system uses configurable thresholds and learning algorithms to minimize false positives while catching genuine spam accounts effectively.
