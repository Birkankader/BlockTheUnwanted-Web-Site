Frequently Asked Questions

Everything you need to know about Block The Unwanted

Is my data safe and private?

Yes. Free tier processing happens entirely locally in your browser with Manifest V3 compliance. Premium tier only sends necessary content to OpenAI for analysis. We never store your personal data or browsing history.

Does it work on both Twitter and X?

Yes, the extension works seamlessly on both twitter.com and x.com domains. It automatically adapts to interface changes.

How does automatic spam detection work?

The extension analyzes user behavior patterns and keyword usage frequency. When an account exceeds configurable thresholds for spam-like behavior, it gets automatically blocked. You can adjust sensitivity settings.

What's the difference between Free and Premium tiers?

Free tier offers local text and basic image processing with Tesseract OCR. Premium tier adds OpenAI GPT-4o powered advanced OCR, video speech-to-text analysis using Whisper, and shareable filter lists.

When will Premium features be available?

Premium features including OpenAI GPT-4o OCR and Whisper video analysis are currently in development. We're working hard to bring you AI-powered filtering capabilities. Follow our updates for launch announcements.

Can I filter specific communities?

Yes, you can block entire Twitter communities by name or link. The extension will hide posts from those communities and their members with intelligent context analysis.

Does it slow down Twitter/X?

No. The extension uses service worker architecture and smart caching for optimal performance. Most users see no noticeable impact on browsing speed.

Can I temporarily disable filtering?

Yes, there's a master toggle in the extension popup that instantly enables/disables all filtering. You can also toggle individual filter types including spam detection.

What languages does the OCR support?

The local Tesseract OCR supports major languages including English, Turkish, Spanish, French, German, and more. Premium tier OCR with OpenAI GPT-4o supports virtually all languages.

Can I export my filter settings?

Yes, you can export your keywords, communities, and settings as a JSON file for backup or transfer to another device. Premium tier will also support shareable filter lists.

Is there a mobile version?

Currently, this is a Chrome desktop extension. Mobile browser extensions have limited capabilities, but we're exploring mobile solutions for the future.
